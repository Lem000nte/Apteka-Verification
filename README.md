# Apteka-Verification
Это бот для верификации, в котором заявки людей отправляются администраторам которые могут одобрить заявку, отклонить или забанить пользователя.

---

## Что умеет бот по функциям?:
- Принимает заявки, направляя их к админам
- Уведомляет админов об новых заявках, с возможностью:
 - Одобрить заявку - кинет одобренному пользователю одноразовую ссылку приглашение
 - Отклонить заявку - Уведомит об отказе
 - Забанить - Запретит писать пользователю заявки до разбана, на случай спама
- Разбан по команде `/unban <ID>`
## Что нужно для того что бы развернуть его?
**1. Клонируйте репозиторий, или скопируйте код**
```
git clone https://github.com/Lem000nte/Apteka-Verification.git
cd tg-request-bot
```
**2. Установите зависимости**
```
pip install python-telegram-bot==20.7
```
**3. Настройте параметры**
- Поставте нужные вам параметры
```
CHAT_ID = -1001234567890 # ID супергруппы
TOKEN = 1234567890 # Токен
ADMIN_IDS = {} # замените на ID админов
```
Что бы получить токен бота, создайте его в @botfather, а что бы получить айдишники используйте @getmyid_bot

**4. Настройте права бота**
Можете ему все права выдать я не уверен че именно ему нужно, не бойтесь там все прозрачно в нём.

**5. Запустите его**
```
python main.py
```

|Команда|Описание|Доступ к команде|
|:---:|:---:|:---:|
|`/start`| Приветсвенное сооб с инструкцией по заявке|Всем|
|`/unban <ID>`|Разбан пользователя по ай ди|Только админам|
